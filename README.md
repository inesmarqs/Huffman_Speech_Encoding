# Huffman Coding Mini-Project

This repository contains the code used for the EE4740 Huffman Coding mini-project.

## Files

- `huffman_speech_compression.ipynb` – Main notebook containing all experiments, analysis, and plots.
- `utils.py` – Helper functions for:
  - Loading speech files
  - Quantisation
  - Huffman codebook construction
  - Encoding and decoding
  - SNR and error-rate calculations
  - Bit-loss simulation
  - RVLC implementation and evaluation
